Untuk Pemakai GPU Radeon, Harap menjalankan program GPU
settings. Karena Tensorflow dan Keras sangat terbatas
dengan GPU dari Radeon

GPU settings akan mengubah cara kerja Tensorflow menjadi
CPU based, mengganti dari default yang memakai GPU based.

Bila tetap ingin memakai GPU, diharapkan untuk mengunduh
Library khusus untuk GPU Radeon.

Untuk Library dateset untuk training tidak ditambahkan
kedalam github karena berisi 1000+ data, yang tentu saja
tidak diperbolehkan oleh github.

yang diberikan oleh discord pada tempat yang sudah diberi
tanda. Harap Berhati-hati dengan Token Discord anda, jangan
sampai terpublikasi !!!
